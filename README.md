# Day-8-Travel-Agency-Page
## 📌 Project Overview
Built a Travel Agency webpage using only semantic HTML5. The page showcases destinations, packages, and top itineraries through clean structure, accessibility basics, and proper document hierarchy.

## 🎯 What I Learned - Day 6/8
Core HTML5 concepts practiced in this build:

**Structure & Semantics**
- Document setup: `<!DOCTYPE html>`, `<html lang="en">`, `<head>`, `<body>`
- Semantic tags: `<main>`, `<section>`, `<figure>`, `<figcaption>`
- Headings hierarchy: `<h1>` to `<h2>` for proper document outline
- Meta tags: `<title>` + `<meta name="description">` for SEO

**Lists & Links**
- Lists: `<ul>` + `<li>` for packages section  
- Anchor tags: external links with `href="https://www.freecodecamp.org/learn"`
- Link behavior: `target="_blank"` to open links in new tab

**Images & Accessibility**  
- Images: `<img>` with `src` + descriptive `alt` attributes
- Figure grouping: `<figure>` + `<figcaption>` for image + caption
- Accessibility: `alt` text for screen readers + SEO
